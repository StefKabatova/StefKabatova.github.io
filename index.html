Project Name: TRIVIA

How it works:

  1. User picks a category to the begin game.

Project Name: Gluten Free Recipes

Using Ruby on Rails framework. I developed a web application that allows user to keep a record of their favorite gluten free recipes.

How it works:

  1. User logs in to account, which directs them to welcome page.
  2. The welcome page displays a list of meal options.Then the user can pick a category to view.
  3. Individual meal category page shows up. It shows available recipes. Also gives user an option to add, edit, and delete recipes.
  4. The user can pick a recipe, which then takes them to it's individual recipe page.
